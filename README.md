### 👨🏻‍💻 关于我 | About Me

你好，我是 **Mr.Q**！欢迎来到我的主页。
我是一名专注于后端开发与系统部署的程序员。深知技术之海浩瀚无垠，始终保持着一颗敬畏和谦虚的心，在代码的世界里慢慢摸索、持续精进。比起华丽的辞藻，我更喜欢在遇到实际问题时去一步步探究底层的答案。

- 🔭 目前主要致力于 **Java** 后端开发，在**企业级业务系统构建、复杂数据流转**以及大数据量报表（如 Excel 导出）优化方面积累了一些实战经验。
- 🌱 最近正在积极探索和学习 **HarmonyOS (鸿蒙)** 开发。
- 🛠 喜欢折腾服务器与网络，对 **Ubuntu**, **Nginx** 反向代理以及内网穿透配置比较熟悉。
- 🤔 遇到问题喜欢探究根本，比如研究底层数据库的源码级安全认证，或是编写自动化脚本来管理运维任务。
- ⚡ 隐藏技能：除了是个能分辨安吉白茶和崂山绿茶的品茶爱好者 🍵，业余时间我还喜欢弹弹吉他 🎸、敲敲电子琴 🎹，没事儿也爱唱唱歌 🎤，用音乐给敲代码的生活加点料。
